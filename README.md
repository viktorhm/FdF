	<h2>build and executing</h2>

	- clone repo
	```sh
	$> git clone git@github.com:viktorhm/FdF.git  cd Fdf
	```

	-add submodule
		```sh
		$> git submodule init git submodule update ```

	-compilation
		```sh
		$> make```

	-how work
		```
		sh $> ./fdf maps/42.fdf ```


	<h2>commande</h2>

	### direction
	> - `w`: up
	> - `s`: down
	> - `a`: right
	> - `d`: left
	### rotation
	> - `{`: angle y+
	> - `}`: angle y-
	> - `>`: angle x+
	> - `<`: angle x-
	### zoom
	> - `-`: -
	> - `+`: +
	### level
	> - `q` :-
	> - `e` :+
