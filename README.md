	<h2>commande</h2>
	### direction
	> - `w`: up
	> - `s`: down
	> - `a`: right
	> - `d`: left
	### rotation
	> - `{`: angle y+
	> - `}`: angle y-
	> - `>`: angle x+
	> - `<`: angle x-
	### zoom
	> - `-`: -
	> - `+`: +
	### level
	> - `q` : -
	> - `e` : +


</ul>
